# BookLoop

Plataforma digital de venta, compra y renta de libros diseñada para fomentar la lectura digital y hacer más accesibles los libros a la comunidad.

---

## Descripción del Proyecto

BookLoop es una aplicación web que permite a los usuarios comprar, vender y rentar libros de manera fácil y segura. Además, cuenta con un panel de administración para gestionar el catálogo de libros. El proyecto fue desarrollado como práctica de programación fundamental por estudiantes, enfocándose en diseño web, maquetación y desarrollo frontend con tecnologías modernas.

---

## Objetivo

Fomentar la lectura digital y promover el acceso a libros físicos a través de una plataforma que permite:
- **Comprar** libros nuevos y usados
- **Vender** libros que ya no necesites
- **Rentar** libros por períodos cortos
- **Administrar** el catálogo de libros (panel admin)

---

## Tecnologías Utilizadas

### Frontend
- **HTML** - Estructura y marcado
- **CSS** - Estilos y maquetación
- **JavaScript** - Lógica interactiva
- **React** - Librería para construir interfaces de usuario
- **Vite** - Herramienta de construcción rápida

### Backend
- **Node.js** - Entorno de ejecución de JavaScript
- **Java** - Creacion de código en parte logica

## Características Principales

- Catálogo de libros con búsqueda y filtrado
